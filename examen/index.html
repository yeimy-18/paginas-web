Matematicas(secundaria)

    1.- Resuelves problemas aritméticos y algebraicos. 2.- Jerarquía de operaciones numéricas. 3.- Realizas transformaciones algebraicas. 4.- Operaciones con polinomios de una y dos variables. 5.- Factorizaciones básicas (incluyendo productos notables y expresiones racionales). 6.- Resuelves ecuaciones lineales. 7.- Sistemas de ecuaciones de 2x2 y 3x3 relacionados con la función lineal. 8.- Modelamiento de situaciones para escribir ecuaciones lineales. 9.- Resuelves ecuaciones cuadráticas. 10.- Ecuaciones cuadráticas de una variable relacionadas con la función cuadrática. 11Modelamiento de situaciones para escribir ecuaciones cuadráticas. 12.- Utilizar ángulos, triángulos y relaciones métricas. 13.- Ejercicios con la aplicación de las propiedades de la suma de ángulos de un triángulo. 14.- Comprendes la congruencia y la semejanza de triángulos. 15.- Ejercicios de aplicación de criterios de congruencia de triángulos. 16.- Problemas de aplicación del teorema de Tales. 17.- Reconocer las propiedades de los polígonos. Resolución de problemas. 18.- Emplear la circunferencia. Ángulos, perímetros y áreas. 19.- Describir las relaciones trigonométricas para resolver triángulos rectángulos. 20.- Sistema sexagesimal y circular. 21Funciones trigonométricas. 22.- Aplicación de razones trigonométricas en triángulos rectángulos. 23.- Aplicas las funciones trigonométricas. 24.- Gráficas de funciones seno, coseno y tangente. 25.- Aplicar las leyes de senos y cosenos. 26.- Aplicación de las leyes en triángulos oblicuángulos. 27.- Aplicar la estadística elemental. 28.- Aplicación de las medidas de tendencia central y de dispersión en datos agrupados y no agrupados. 29.- Emplear los conceptos elementales de la probabilidad. 30.- Aplicación de las leyes aditiva y multiplicativa de las probabilidades. 31.- Aplicas las propiedades de segmentos rectilíneos y polígonos. 32.- Distancia entre dos puntos. 33.- Perímetro y área de polígonos. 34.- Aplicas los elementos de una recta como lugar geométrico. 35.- Pendiente y ángulo de inclinación de una recta. 36.- Paralelismo y perpendicularidad. 37.- Utilizas distintas formas de la ecuación de una recta. 38.- Aplicación de las distintas formas de ecuación de una recta. 39.- Aplicas los elementos y las ecuaciones de una circunferencia. 40.- Aplicación de las distintas formas de ecuación de una circunferencia. 41.- Aplicas los elementos y ecuaciones de la parábola. 42.- Aplicación de las distintas formas de ecuación de una parábola. 43.- Aplica los elementos y las ecuaciones de la elipse. 44.- Aplicación de las distintas formas de 45.- ecuación de una elipse. 46.- Reconocer y realizar operaciones con distintos tipos de funciones. 47.- Funciones y Relaciones. 48.- Dominio y Rango. 49.- Reglas de correspondencia. 50.- Aplicar funciones especiales y transformaciones. 51.- Función inversa, función escalonada, función valor absoluto, función identidad y función constante. 52.- Empleas funciones polinomiales de grados 0, 1, 2, 3 y 4. Reconocimiento del patrón de comportamiento gráfico de las funciones polinomiales. 53.- Aplicas funciones racionales. Asíntotas horizontales y verticales. 54.- Reconocimiento del patrón de comportamiento gráfico de las funciones racionales. 55.- Aplicas funciones exponenciales y logarítmicas. 56.- Gráficas de las funciones exponencial y logarítmica. 57.- Aplicas funciones periódicas. Representación gráfica de las funciones seno y coseno. 58.- Características: amplitud, frecuencia y periodo.
